Hakkımda

Yazılım işini tutkuyla yapan biriyim. Yazılım maceram 2014 yılında lisede C++ ve Visual Basic yazarak başladı. 2019 yılında üniversiteye başladım. Çeşitli etkinliklere katılım sağladım. Bunlardan en heyecan verici Google'unda ortaklığı ile düzenlenen İstanbul Startupweekend Hackathonu'ydu. Ekipçe yarışma ikinciliğini elde ettik. Üniversite süresince MVC teknolojisini kullanarak Donation Management Systems projesini geliştirdim. Şuan gelişimimi sürdürmekteyim. ASP.NET Core ve Web API üzerinde kendimi geliştirmeye devam ediyorum. Aynı zamanda bir sivil toplum kuruluşunda gönüllülük yapıyorum.
